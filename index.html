13 Principles of Display Design Applied

1. Pressure Sensor Detection

Weight sensors in rear seats detect child/car seat presence even when the driver cannot physically see the back seat due to seat position and height limitations — directly solving Amin's visibility obstruction.

Redundancy Gain

2. Multi-Stage Escalation

Unlike the original single low-intensity chime, the redesign escalates through 4 stages: dashboard visual → horn pulse → phone alert → emergency call. Each stage increases urgency if no response.

Discriminability

3. Cabin Temperature Monitor

Real-time interior temperature display turns red when dangerous. Replaces mental estimation with a direct visual readout — reducing cognitive load on a stressed/distracted parent.

Replace Memory with Visual

4. Bird's-Eye Seat Map

Top-down cabin view shows which seats are occupied using colour coding (green = driver, red = unattended passenger). Provides spatial awareness the driver lacks due to physical obstruction.

Pictorial Realism

5. Phone Push + SMS Alert

When the driver walks away and the child is still detected, an automatic notification is sent to their phone. This multi-modal approach (auditory + visual + haptic) breaks through cognitive tunnel vision.

Multiple Resources (Wickens)

6. Function Allocation

Machine handles: weight detection, temperature monitoring, alert escalation, phone notification, emergency calling. Human handles: final decision to return to the vehicle. This offloads memory and vigilance tasks to the system — the exact cognitive functions that failed due to Amin's high cognitive load and routine change.
